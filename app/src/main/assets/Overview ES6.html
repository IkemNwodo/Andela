This module introduces the fundamental concepts and new features in ES6(EcmaScript2015).

Learning Objectives

By the end of this module, you should be able to understand:

  1. Have a basic understanding of the structure of ES6.

  2. Should be able to adhere to ES6 syntax conventions.

  3. Should be able to write simple ES6 applications.

  4. Should be familiar with the new JS features introduced in ES6.
